# modify-code スキル

[rules.md](rules.md) に定義されたコーディング規約を読み込み、指定されたコードを規約に沿って修正する。

## 手順

1. [rules.md](rules.md) を読んでコーディング規約を把握する
2. `$ARGUMENTS` で指定されたファイルを読む（指定がない場合はIDEで開いているファイルを対象にする）
3. 規約違反の箇所を特定する
4. Edit ツールで修正する
5. 修正した箇所と理由を箇条書きで報告する

## 注意

- 規約に違反していない箇所は変更しない
- ロジックの変更は行わない（スタイル・命名・構造の修正のみ）
- 修正できない箇所があれば、その理由を明記する
